MovieDB

Unidirectional Association App

  • People
  • Movies

This app supports the following operations:

  • Manage person data
  • Manage movie data

For playing with the app, you may first want to and then view it (e.g. by choosing "Manage movies" and then "Retrieve and list all movies"). For observing the internal status and error messages, turn on the JavaScript console.

This JavaScript front-end data management app takes care of the object types Person and Movie as well as the two unidirectional associations that assign a director and (one or more) actors to a movie. The app supports the four standard data management operations (Create/Retrieve/Update/Delete). It extends the Validation App by adding code for handling unidirectional associations.

This site is product of @medinger's work during summer semester 2023 as part of the module "Designing and Implementing Web Applications" from the Chair of Internet Technology, Brandenburg University of Technology, Germany.
